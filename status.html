2. Was fehlt für MVP

20 Arbeitspakete in 4 Kategorien. MVP-Deadline: KW27 (06.07.2026, Abteilungsleitertagung).

Pricing & Kalibrierung ~8 Tage
1. Schieberegler Listenpreis / Marktpreis / Ausschreibung
PoC berechnet LPV-Listenpreise. Reale Faktura ist -46% bis -85% niedriger (aus Versand-Daten). Ohne Marktpreis-Faktor gibt das Tool Preise aus, die 2-3× höher sind als der Markt. #1 Blocker für Nutzbarkeit.
2. Kriterien Preisfindung (S. Pausch)
3 Faktoren: Anlagenmerkmale (60-90%), Dokumentation (8-15%), Ortskenntnis (2-25%). Skaliert nicht-linear mit BGF. Einbauen als Rückfragen + Confidence-Modifikatoren.
3. Volkswohlbund-Kalibrierung / Rahmenvertrag-Modell
Reale Preise: UV PLV 270€ vs EP 101€ (-63%). Rahmenvertrag-Rabattmodell für wiederkehrende Kunden.
4. Staffel-Kalibrierung Blitzschutz
57234_Versand Erlös-h-Satz Distribution vs. aktuelle Staffeln (33/30/28/26/24€). Schwellen-Prüfung: BSK 40€ vs 45,50€, Garage ≤25 vs ≤30 SP (Klärung mit S. Pausch).
Workflow & Output ~6 Tage
5. Angebots-Export (PDF / Textbausteine)
S. Feith-Request: Output zum Kopieren in SAP/Angebot. PDF mit Positionen, Textblöcke. Ohne Export ist das Tool ein Kalkulator, kein Workflow-Tool.
6. Multi-Gebäude-Angebot
S. Feith: "Schule + Turnhalle + Sporthalle = 3 Anlagen, 1 Angebot." Aktuell: Single-Anlage pro Anfrage. Benötigt: Anlagen-Liste → Gesamtangebot.
7. Strukturiertes Formular (Alternative zu Chat)
Drop-downs, Slider, Eingabefelder. Sachverständiger weiß was er ausfüllen muss — Chat ist für Neukunden, Formular für Profis.
8. Schieberegler grob/mittel/fein
S. Feith-Request: grob = nur BGF + Sicherheitsfaktor, fein = alle Merkmale. Chat-Flow unterstützt das implizit, UI-Toggle fehlt.
Daten & Qualität ~5 Tage
9. MA507 Versand-Daten (DGUV V3)
Golden Set aktuell nur Gersthofen (40 Anlagen). Mit Versand: tausende Einträge für echte Kalibrierung.
10. Inspektionspflichten-Lookup
70 Einträge (netinformRE): Regelwerk × Gebäudeart → Prüfzyklus + Qualifikation. Chat: "Ihre nächste Prüfung ist in X Jahren fällig."
11. Kombi-Begehung Rabatt
Blitzschutz + RLT gleichzeitig → nur 1× Reisekosten. Cross-Product-Rabattlogik.
12. CO-Warnanlagen (Garage-Ergänzung)
Kalkulationshilfen hat eigene Sektion "CO-Warnanlagen in Garagen" (510€ + 242€ + 239€). Garage-Kalkulation ist ohne das unvollständig.
13. Conversation History / Session Persistence
Sessions sind aktuell in-memory (gehen bei Restart verloren). PostgreSQL-backed Sessions für "gestern berechnetes Angebot nochmal öffnen".
UX & Design ~6 Tage
14. TÜV Corporate Design
Figma-basiertes Design System: Farben, Fonts, Logo, Komponenten.
15. Responsive / Tablet
Sachverständige sind im Außendienst mit iPad. Aktuell Desktop-only.
16. Vergleichsansicht / Ähnliche Anlagen
"Letzte Woche hatte ich ähnliches Gebäude — was war der Preis?" Similar-Anlagen Lookup aus Golden Set.
Infrastruktur & Betrieb ~5 Tage
17. User Auth / Rollen
Abteilungsleiter vs Sachverständiger vs Verwaltung. Unterschiedliche Sichten und Berechtigungen.
18. Admin-Panel (LPV-Pflege)
TÜV aktualisiert Preise/Staffeln/LPV jährlich. Ohne Admin-Panel: Code-Änderung nötig. Mit: Self-Service.
19. Monitoring / Analytics Dashboard
Wie viele Angebote generiert, welche Produkte am meisten genutzt, durchschnittlicher Angebotspreis. Dashboard für S. Feith.
20. API für SAP-Anbindung (Dokumentation)
REST-Endpoints dokumentiert für zukünftige S4/HANA-Integration (2029). Swagger/OpenAPI Spec.
